# Spring Cloud Gateway

Spring Cloud Gateway 학습 프로젝트

## 학습 내용

1. YML 기반 API 라우팅 설정
2. Java 코드 기반 API 라우팅 설정
3. 인증 필요 시, 커스텀 필터를 통한 인증 처리 후 라우팅
4. 글로벌 예외 처리 구현
5. WireMock 사용 테스트 구성
